Kaleidoscope

Work
Generative Art
Year
2025
Instructor
Jinsung Yoo
Selected
2025 Emerging Media Showcase, 2026 Emerging Media Showcase

An interactive kaleidoscope uses hand tracking to detect the positions of the thumb and index finger, as well as the distance between them. This distance is then used as a variable to generate dynamic visuals. Through this process, the project creates evolving graphics in real time using generative coding.
